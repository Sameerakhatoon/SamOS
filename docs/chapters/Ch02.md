# Ch 2 - Requirements

**Book pages:** 5 (Part 1, Section 2)
**Code in this chapter:** none - prose

## The book's four requirements

1. **C programming** - syntax, control flow, pointers, manual memory management.
2. **Ubuntu (or equivalent Linux)** as the dev host. The book uses Ubuntu; a VM is fine.
3. **Basic assembly familiarity** - what registers/segments/instructions do; we'll grow this as we go.
4. **A curious mind** - the book's call-out: kernel work rewards stubborn curiosity over raw talent.

## How SamOs satisfies these

- Dev host: WSL2 Ubuntu (this `Ubuntu-fresh` distro).
- C: settled. NASM for assembly.
- Tooling install gets its own chapter when the book covers it.
