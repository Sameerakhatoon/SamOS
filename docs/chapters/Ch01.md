# Ch 1 - Introduction

**Book pages:** 3-4 (Part 1, Section 1)
**Code in this chapter:** none - prose

## What the book promises

A walk through building a multitasking kernel + FAT16 filesystem driver from scratch. Real Mode first (boot loader, BIOS disk reads, real-mode interrupts), then Protected Mode (memory protection, paging, VFS, FAT16, tasks/processes, heap, keyboard driver). The book is the companion to McCarthy's video course of the same name.

Prerequisites: working C + a little assembly. The book promises to teach the rest.

## What we take away

- Trajectory: Real Mode → Protected Mode → 32-bit multitasking kernel with FAT16 + ELF user-space.
- Goal end-state: an OS that loads ELF user programs, runs them in ring 3, multitasks via PIT preemption, and exposes a syscall ABI.
- SamOs deviation from the book: we'll bump to **64-bit (long mode)** once the book's protected-mode chapters establish the foundations. The 64-bit upgrade gets its own chapter note when we make the jump.
