# Ch 3 - Who This Book Is For

**Book pages:** 6 (Part 1, Section 3)
**Code in this chapter:** none - prose

## Audience

- Aspiring OS developers.
- CS students wanting deeper-than-coursework OS internals.
- Working programmers exploring a new domain.
- Hobbyists drawn to "how does an OS actually work?".
- Educators teaching OS-construction courses.

## Tone-setting

The book is hands-on; concepts are introduced just-in-time with code. We mirror that here - chapter notes are tight on theory and rely on the implementation + tests for the real understanding.
